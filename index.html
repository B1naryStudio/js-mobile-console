js-mobile-console

That's how you install it:

npm install js-mobile-console
		
bower install js-mobile-console

That's how you integrate it:

	mobileConsole.show();
	console.log(1);

	asdf;
	
	mobileConsole.commands({
		'check': 'var a = 10; a;',
		'run': '10/5'
	});
or 
	mobileConsole.options({
		showOnError: true,
		proxyConsole: true,
		isCollapsed: true,
		catchErrors: true
	});

	console.log(1);
	asdf;
